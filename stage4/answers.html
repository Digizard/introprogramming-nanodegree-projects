Reverse Engineering Project

Directions: Identify which type of programmer(Front End, Back end, Mobile Programmer or Data Analyst) would work on the features of Project Compass. Type your answers between the paragraph tags and be sure to provide an brief explanation for your choice.

1. Which programmer would build and manage the database for storing map pin locations in Project Compass?

Back End

Working with databases is one of the areas that back end developers specialize in.

2. Which programmer would make Project Compass responsive on different screen sizes on the web?

Front End

This involves the appearance of a webpage and how the page works across multiple devices and screen sizes.

3. Which type of mobile programmer (Android or iOS) would be using Swift to turn Project Compass into a mobile app?

iOS

Objective C and Swift are the programming languages used for developing iOS applications. Android applications can use Java, C, and C++.

4. Which programmer would add the image to Project Compass?

Front End

This involves putting the image on the page visibly, which has to do with the appearance of the site.

5. Who would evaluate information about Project Compass users to give insights for product decisions?

Data Analyst

The finding of relationships and patterns in data is what data analysts do.

6. Who would make requests to the Google Maps API in Project Compass?

Front End

This involves making requests to other internet sources for information, as opposed to actually building and providing said information. Front end developers work with APIs while back end developers actually build them.

7. Which programmer would need to consider hardware components to make Project Compass usable with a microphone or camera?

Mobile

Using the microphone and camera on a phone is rather common for mobile applications, especially since these devices come with them built in.

8. Which programmer would make a post request endpoint to add comments onto Project compass?

Back End

Post requests happen behind the scenes. It is the interaction between client and server, and it affects the data stored on the server.

9. Which programmer would look at user clicking rates on Project Compass to make recommendations for external tourism sites?

Data Analyst

Based on relationships and patterns seen, a recommendation is being made. That is a decision based on gathered data.
